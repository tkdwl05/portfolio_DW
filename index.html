시계열 · 정형LG Aimers 해커톤 817팀 중 138등 · 상위 16.9%

식음업장 메뉴 수요 예측 (LG Aimers 온라인 해커톤)

리조트 식음업장의 영업장–메뉴 조합별 향후 7일 수요를 예측하는 시계열 + 표형(tabular) 문제 · 평가 지표 SMAPE · 1,750명 참가, 5인 팀(NOLSTM)

STACK
CatBoost LightGBM XGBoost Prophet Optuna AutoGluon-TS SLSQP GPU
핵심 전략
  • 피처: 28일 룩백 롤링(mean·std·last·weekday), 공휴일·"끼인 평일" 라벨, 외부데이터 결합(날씨 ASOS/AWS·축제·검색량·SNS 버즈)
  • 모델: CatBoost/LGBM/XGB 다중출력 앙상블, count 데이터에 Poisson 목적함수 + 커스텀 SMAPE
  • 앙상블: SLSQP로 합=1 제약 하 블렌딩 가중치 최적화(단순 평균 대비 차별점) + 예측 반올림
결과
  • 817팀 중 138등 (상위 16.9%) · Private SMAPE 0.5187
  • baseline 0.711 → 약 27% 개선 · 1위 0.437(상위권 0.437~0.445)과는 격차 있음 · 본선 미진출
기여·한계
5인 팀 중 전처리·피처 / 부스팅·SLSQP 블렌딩 / AutoGluon 담당(LSTM은 팀원 — 팀명 "NOLSTM"의 유래). 한계: 검증에 랜덤 분할을 써 시계열 data leakage 위험 → 다음엔 시간 기반 분할(rolling-origin)로 개선 예정.

가속 MRI 영상 복원 — ViT 인코더 + Mamba/GRU 디코더

fastMRI brain multicoil의 R=4 언더샘플 k-space에서 영상을 복원하는 솔로 연구 · ViT 인코더에 디코더만 GRU(ETER) vs Mamba(SS2D)로 교체해 통제 비교 · 평가 지표 SSIM/PSNR

STACK
PyTorch ViT Mamba (SS2D) Bi-GRU (ETER-Net) SSIM / PSNR wandb
접근
  • 공통 ViT 인코더 + 디코더만 GRU vs Mamba로 교체해 아키텍처 변수를 통제한 정밀 비교 설계
  • R=4 equispaced 언더샘플링 · L1 + (1−SSIM) 손실 · 8GB GPU 제약 하 gradient checkpointing
결정적 인사이트
커스텀 SSIM 평가 버그 발견·수정 — val_range 처리로 SSIM이 인위적으로 낮게 측정되던 것을 skimage 표준으로 재평가하자 0.7227 → 0.8584로 도약하고 아키텍처 랭킹까지 반전. "측정 도구를 먼저 검증한다"를 실증.
결과
  • 320 트랙: SS2D-ViT full-eval SSIM 0.8924 vs pretrained U-Net 0.8858 — 훨씬 작은 모델로 SSIM 추월(PSNR 마진은 작음, 정직 표기)
  • 384 트랙: ViT-Base(114.8M) ETER/SS2D near-tie SSIM_m ~0.908 · composite 0.9127
  • TTA·앙상블은 시도했으나 임계 미달로 모두 negative(정직)
기여·상태
솔로. 본인 = ViT↔GRU/Mamba 융합 디코더 · R=4 데이터 파이프라인 · 학습/평가/ablation 인프라 · 평가 버그 수정(ViT·GRU·Mamba·MAE 코어는 외부). 진행 중 — DC 블록·blur 완화가 다음 과제.

시계열 · 딥러닝기상청 빅데이터 콘테스트 3-Fold RMSE 0.99–7.97

119 신고 건수 예측 — 기상 데이터 융합 LSTM

부산 소방청 119 신고 건수를 기상 데이터와 결합해 단기 예측 · 지점별 시계열 특성을 반영한 LSTM 회귀 모델

STACK
TensorFlow / Keras scikit-learn pandas 시계열 전처리 교차검증 Matplotlib / Seaborn
전처리
  • 기상(ASOS/AWS 2020–2024)과 소방(call119, cat119) 데이터를 지점·날짜/시간 키로 병합
  • 결측값 제거/대체, 기상 데이터 보간 및 이상치 탐지
  • 스케일링: StandardScaler(기본), RobustScaler(이상치 내성 실험)
  • 시계열 윈도잉: lookback=3 → 다음 시점 예측, 입력 형태 (samples, timesteps=3, features=7)
모델 구조
2층 LSTM + Dense(1) · cell_size 32/64 실험 · Adam(lr=0.001) · MSE loss · 500 epochs
성과
  • 11개 소방서(지점)별 맞춤형 예측 모델 구축
  • 지점별 3-Fold 평균 RMSE 0.99 ~ 7.97 달성 — 921번 지점 0.99로 최우수
  • RMSE 분포·히트맵·랭킹·종합 대시보드로 성능 시각화
배운 점
다변수 시계열 LSTM 모델링, 지점별 특성 차이 분석, 기상–사회 데이터 융합. 향후 개선 방향으로 TimeSeriesSplit·EarlyStopping 도입, Attention/GRU/Transformer 비교, 부스팅과의 앙상블을 도출.
그래프 딥러닝Kaggle TM-Score · Best-of-5

Stanford Ribonanza RNA Folding 2 — RNA 3D 구조 예측

RNA 염기 서열에서 각 잔기 C1' 원자의 (x,y,z) 좌표를 예측하는 Kaggle 경진대회 · 평가 지표 TM-Score(Best-of-5) · 회전·이동에 불변한 예측 파이프라인을 직접 설계

아키텍처
  • EGNN(동변성 GNN) 기반으로 좌표의 회전·이동에 불변한 예측 구조 설계
  • 염기쌍 확률(A-U · C-G · G-U wobble)을 그래프 edge 특징으로 활용
  • 다중 헤드 좌표 예측 + confidence 추정, softmin 기반 best-of-5 근사
  • 다층 손실: Kabsch RMSD + edge-distance + repulsion + variance matching + confidence
  • EMA 타깃 + centering 기반 self-supervised consistency 정규화(DINO 계열 아이디어 차용)
  • TBM(Template-Based Modeling) 하이브리드 추론 결합
엔지니어링
  • 혼합정밀도 학습(autocast/GradScaler), 손실은 FP32로 계산해 안정화
  • Kabsch SVD Tikhonov 정규화·nan_to_num·tanh 좌표 제한으로 NaN/Inf 방지
  • 20+ 버전(v2 → v26) 반복으로 손실·아키텍처를 체계적으로 개선
배운 점
동변성 그래프 신경망, 자기지도 아이디어를 지도학습의 auxiliary loss로 결합하는 설계, 대규모 파이프라인의 수치 안정성과 시간 예산 관리.

데이터 전처리 · 피처 엔지니어링

  • 이상치 — STL 분해, Z-score, IQR, Isolation Forest, Tweedie GLM, ADTK 등 복수 기법 비교 적용
  • 결측·보간 — linear/spline/time/rolling mean 비교, RMSE 기준 최적 방식 선택
  • 스케일링 — StandardScaler / RobustScaler 상황별 선택
  • 캘린더·시계열 피처 — 공휴일·주말·"끼인 평일" 라벨링, 롤링 통계, 추세·계절 피처, 이동평균 대비 비율
  • 외부 데이터 결합 — 기상청 API, 네이버 데이터랩, 공공데이터포털(상권/유동인구), 행사·축제 일정
  • 데이터 검증 — 누락 날짜 점검, 0 매출 vs 결측(NaN) 구분, 단위/로그 변환 필요성 확인

정형 데이터 모델링

  • 부스팅 3대장(XGBoost/LightGBM/CatBoost)의 특성 이해와 GPU 가속 활용
  • 다중출력 예측, 목적함수 커스터마이징(SMAPE, Tweedie/Poisson)
  • 앙상블·블렌딩(SLSQP 가중치 최적화), Optuna 하이퍼파라미터 탐색
  • SHAP 기반 피처 중요도 분석

시계열 예측

  • 베이스라인(Naive, Seasonal Naive, SARIMAX, Prophet) → 부스팅 → 딥러닝(N-BEATS, LSTM) 비교 체계
  • 시계열 교차검증(TimeSeriesSplit)으로 data leakage 방지
  • horizon별(1·2·3일) 성능 저하 패턴 분석

딥러닝 아키텍처

  • 시퀀스 — LSTM/GRU, Attention, Transformer
  • 그래프 — GNN, EGNN(동변성), pair-aware message passing
  • 자기지도 정규화 — EMA 타깃 + centering 기반 consistency (DINO 계열 아이디어 차용)
  • 3D 좌표 예측 — pair 표현, confidence 추정, best-of-K 예측
  • 역문제·복원 — ViT 인코더 + Mamba(SS2D)·Bi-GRU 디코더, R=4 k-space 언더샘플링 MRI 복원(SSIM/PSNR)

LLM · 효율적 학습
