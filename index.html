식음업장 매출 수량 예측

각 영업장–메뉴 조합에 대해 향후 7일간의 매출 수량을 예측하는 시계열 + 표형(tabular) 문제 · 평가 지표 SMAPE

STACK
CatBoost LightGBM XGBoost Prophet Optuna PyTorch (LSTM) AutoGluon GPU 가속
전처리
  • 한국 공휴일(2023–2025) + 주말 + "끼인 평일"을 is_holiday로 라벨링
  • 이상치 탐지: STL, Z-score, IQR, Isolation Forest, Tweedie GLM, ADTK(Threshold/Seasonal) 비교 적용
  • 보간: linear / spline / time / rolling mean 비교 후 RMSE 최소 방식 채택, 평일·휴일 분리 처리로 계절성 분리
  • 공휴일·요일·행사(화담숲 오픈 여부) + 업장 더미/집계 피처를 train/test에 결합
  • 부스팅용 특성: 룩백 28일 / 예측 7일, 롤링 통계(mean·std·last), 요일·공휴일·채널별 통계
모델링
  • CatBoost(MultiRMSE) + LightGBM/XGBoost 다중출력 래퍼 앙상블
  • SLSQP 기반 가중치 최적화로 합 제약(=1) 하에 SMAPE를 최소화하는 블렌딩 가중치 탐색
  • Prophet 예측을 피처로 주입한 버전 / 미포함 버전 각각 실험
  • Optuna로 CatBoost(iterations·depth·lr) 탐색, XGBoost/LightGBM은 Tweedie·Poisson 목적함수 실험
  • 커스텀 SMAPE 목적함수 직접 구현
배운 점
다중출력 부스팅 앙상블 설계, 목적함수·손실 커스터마이징, 최적화 기반 블렌딩 가중치 탐색 경험.

119 신고 건수 예측 — 기상 데이터 융합 LSTM

부산 소방청 119 신고 건수를 기상 데이터와 결합해 단기 예측 · 지점별 시계열 특성을 반영한 LSTM 회귀 모델

STACK
TensorFlow / Keras scikit-learn pandas 시계열 전처리 교차검증 Matplotlib / Seaborn
전처리
  • 기상(ASOS/AWS 2020–2024)과 소방(call119, cat119) 데이터를 지점·날짜/시간 키로 병합
  • 결측값 제거/대체, 기상 데이터 보간 및 이상치 탐지
  • 스케일링: StandardScaler(기본), RobustScaler(이상치 내성 실험)
  • 시계열 윈도잉: lookback=3 → 다음 시점 예측, 입력 형태 (samples, timesteps=3, features=7)
모델 구조
2층 LSTM + Dense(1) · cell_size 32/64 실험 · Adam(lr=0.001) · MSE loss · 500 epochs
성과
  • 11개 소방서(지점)별 맞춤형 예측 모델 구축
  • 지점별 3-Fold 평균 RMSE 0.99 ~ 7.97 달성 — 921번 지점 0.99로 최우수
  • RMSE 분포·히트맵·랭킹·종합 대시보드로 성능 시각화
배운 점
다변수 시계열 LSTM 모델링, 지점별 특성 차이 분석, 기상–사회 데이터 융합. 향후 개선 방향으로 TimeSeriesSplit·EarlyStopping 도입, Attention/GRU/Transformer 비교, 부스팅과의 앙상블을 도출.

3D 좌표 예측 — 동변성 그래프 신경망(EGNN)

구조화된 입력으로부터 3D 좌표를 예측하는 Kaggle 경진대회 · 회전·이동에 불변한 예측 파이프라인을 직접 설계

접근
  • EGNN(동변성 GNN) 기반으로 좌표의 회전·이동에 불변한 예측 구조 설계
  • DINO/SwAV 스타일 self-distillation(자기지도학습)을 결합한 학습 파이프라인
  • 다중 헤드 좌표 예측 + confidence 추정, best-of-K 근사
  • 혼합정밀도(FP16) 최적화, 시간 예산 기반 조기 중단으로 대회 시간 제한 대응
  • 핵심 연산을 순수 Python 반복문에서 np.einsum 벡터화로 약 100배 가속
배운 점
동변성 그래프 신경망, 자기지도학습을 지도학습의 auxiliary loss로 결합하는 설계, 대규모 파이프라인의 수치 안정성과 시간 예산 관리.

기여

  • 각 연구와의 차별점·개선점·적용 방안 도출
  • 종합설계 발표 자료(PPT 아웃라인 + 대본)를 영어·한국어 두 버전으로 기획

딥러닝 · 데이터사이언스 지식

KNOWLEDGE

데이터 전처리 · 피처 엔지니어링

  • 이상치 — STL 분해, Z-score, IQR, Isolation Forest, Tweedie GLM, ADTK 등 복수 기법 비교 적용
  • 결측·보간 — linear/spline/time/rolling mean 비교, RMSE 기준 최적 방식 선택
  • 스케일링 — StandardScaler / RobustScaler 상황별 선택
  • 캘린더·시계열 피처 — 공휴일·주말·"끼인 평일" 라벨링, 롤링 통계, 추세·계절 피처, 이동평균 대비 비율
  • 외부 데이터 결합 — 기상청 API, 네이버 데이터랩, 공공데이터포털(상권/유동인구), 행사·축제 일정
  • 데이터 검증 — 누락 날짜 점검, 0 매출 vs 결측(NaN) 구분, 단위/로그 변환 필요성 확인

정형 데이터 모델링

  • 부스팅 3대장(XGBoost/LightGBM/CatBoost)의 특성 이해와 GPU 가속 활용
  • 다중출력 예측, 목적함수 커스터마이징(SMAPE, Tweedie/Poisson)
  • 앙상블·블렌딩(SLSQP 가중치 최적화), Optuna 하이퍼파라미터 탐색
  • SHAP 기반 피처 중요도 분석

시계열 예측

  • 베이스라인(Naive, Seasonal Naive, SARIMAX, Prophet) → 부스팅 → 딥러닝(N-BEATS, LSTM) 비교 체계
  • 시계열 교차검증(TimeSeriesSplit)으로 data leakage 방지
  • horizon별(1·2·3일) 성능 저하 패턴 분석

딥러닝 아키텍처

  • 시퀀스 — LSTM/GRU, Attention, Transformer
  • 그래프 — GNN, EGNN(동변성), pair-aware message passing
  • 자기지도학습 — DINO/SwAV 스타일 self-distillation (EMA teacher, centering, sharpening)
  • 3D 좌표 예측 — pair 표현, confidence 추정, best-of-K 예측

LLM · 효율적 학습

  • QLoRA + gradient checkpointing + Unsloth + bf16 + Flash Attention 2 조합으로 제한된 VRAM(8GB)에서 7B급 파인튜닝
  • 소형 모델(Llama 3.2, Qwen2.5, Phi-3.5, EXAONE, Kanana 등) 비교·선택 기준 이해
  • Mixed precision, 시퀀스 길이 제한, 시간·메모리 예산 관리
